
# Football Analytics Dashboard

Interactive Football Analytics Dashboard powered by StatsBomb Open Data.

## Features

- Team Tournament Analysis
- Team Match Analysis
- Team Comparison
- Player Tournament Analysis
- Player Match Analysis
- Player Comparison
- Similar Teams
- Similar Players

## Tech Stack

- Streamlit
- Python
- Pandas
- NumPy
- Matplotlib
- mplsoccer
- StatsBomb Open Data
